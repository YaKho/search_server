# C++ Search Server
Реализация поискового сервера. Обеспечивает поиск по документам.
## Возможности
- Ранжирование результатов поиска по TF-IDF.
- Поддержка стоп-слов и минус-слов.
- Обработка через очередь запросов.
- Возможность параллельной работы.
## Требования
- C++17 и выше.
